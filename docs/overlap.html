Calvin Program Overlap Viewer

Visualize program and core overlap

This bookmarklet runs on any Calvin catalog program page and opens a side panel showing direct course overlap with Core requirements.

Program overlap viewer Open a program page Core annotator bookmarklet

Drag the bookmarklet button into your bookmarks bar, then click it while viewing any program page.

Install

  1. Show your browser bookmarks bar.
  2. Drag Program overlap viewer into the bar.
  3. Open any page with path preview_program.php on catalog.calvin.edu.
  4. Click the bookmark to load the viewer panel.
