Calvin Core Annotator

Install the bookmarklet once, then update it by pushing code.

This bookmarklet loads the latest annotator script from this GitHub Pages site each time you click it on the Calvin catalog core-program page.

Calvin Core Annotator Open the catalog page

Desktop browsers usually let you drag the bookmarklet link straight into the bookmarks bar.

Install

  1. Show your browser bookmarks bar.
  2. Drag the Calvin Core Annotator link above into the bar.
  3. Open the Calvin catalog core-program page.
  4. Click the bookmark you just added.

If drag-and-drop is awkward, create a new bookmark manually and paste the URL below into the bookmark URL field.

Use

  1. In Workday, open Academic Record, then Academic History.
  2. Select all of the transcript content and copy it.
  3. On the catalog page, click the bookmarklet.
  4. Paste the transcript into the panel that appears above the core requirements.

Notes

  • Because the bookmarklet fetches bookmarklet.js from this site, code updates take effect after you deploy GitHub Pages. The bookmark itself does not need to change.
  • The browser extension path still works for local testing.
  • This tool is unofficial and should be treated as a planning aid, not an official degree audit.
